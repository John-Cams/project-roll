Hi I'm AriMaexisted and this is my game Project_Roll!

##The current version of the game can be played [right here] (https://arimaexisted.itch.io/project-roll)

This is a very simple game involving cards and dice!
First you place the cards into the right, each card correlating to a different effect and each slot correlating to a different side.
When submit is pressed, the dice is rolled, and the winner is determined!

This game was made in Godot version 4!
